I maintain the following Open Source projects. Most of them are written in Lua and MIT licensed.

If you report an issue with one of them, I will do my best to fix it. If you use them in production, I would certainly appreciate if you

told me

.

  • Lua Toolbox: website to discover, choose and recommend Lua modules
  • fakeredis: Redis mock for Lua with the same interface as redis-lua
  • Haricot: Beanstalk client for Lua
  • cwtest: my reinvention of the "test helper" wheel
  • lua-mirrorfs: Lua module that implements a FUSE mirroring a directory, and a starting point to write other FUSE filesystems in Lua
  • luajit-msgpack-pure: MessagePack implemented in pure LuaJIT
  • lua-chacha: Lua C module implementing the ChaCha stream cipher
  • lua-http-digest: client-side HTTP Digest Authentication for LuaSocket
  • lua-multipart-post: Lua HTTP Multipart Post helper that does just that
  • tolog: simple C program to make software that logs to standard output play nice with logrotate
  • localua: Bash script to build a self-contained Lua and LuaRocks in a directory

For more, see my GitHub account.
